# Project Context: [lotto_app]

## 📋 專案概述

**專案名稱**: 中文:好運自己選  英文:lotto_app
**專案類型**: 台灣彩券選號分析應用程式  
**開發語言**: Python + Kivy  
**目標平台**:Windows/ Android (Google Play) / iOS (Apple Store)  
**專案簡介**
1. 這是用python寫的彩券程式，不同於一般彩券程式，提供對獎功能，本程式是提供自由選號介面，並依所選的任意球數球號，找出在歷史獎號中曾經出現的期別並做次數、獎別統計，供使用者做為自選號的參考。
2. 提供自由選號並分析的台灣彩券種類包括威力彩、大樂透、今彩539、三星彩及四星彩。
3. 每一彩種都提供選號查詢、查重複獎號、自選號中獎詳情、儲存自選號及提取自選號等功能(已完成)。
4. 自動連線Supabase更新開獎獎號(Windows已完成)。
5. 檢查APP版本並更新(未完成)。
6. 加入Google AdMob廣告平台(未完成)。
7. 本專案先做Windows版(已完成主要功能)並測試，再轉 Android (Google Play) / iOS (Apple Store)(未完成)。

## 🎯 核心需求

### 1. 主要功能需求(Windows已完成)

#### 1.1 彩券選號系統
- **威力彩選號**: 第一區1-38最多選6個，第二區1-8最多選1個。
- **大樂透選號**: 1-49最多選6個。
- **今彩539選號**: 1-39最多選5個。
- **三星彩選號**: 佰位、拾位、個位各選0-9，不必選滿三位數。
- **四星彩選號**: 仟位、佰位、拾位、個位各選0-9，不必選滿四位數。

#### 1.2 選號查詢分析
- 可自行選取不同號數彩球數個(依不同彩種球數限制，不超過球數限制即可，不需選滿球數)。
- 使用者所選的號碼叫做自選號。
- 根據使用者的自選號，查詢歷史開獎記錄lotto_history.db。
- 查詢並列出所有包括自選號的中獎期別、開獎日期、獎號(自選號以不同顏色顯示)及獎別。
- 顯示中獎總筆數及按獎別統計中獎次數。
- 提供排序功能（日期升序/降序）。

#### 1.3 自選號管理
- 儲存自選號碼組合(比較特別的是，威力彩可僅選滿一區就可儲存自選號，不一定要連第二區都選才可以儲存，其餘彩種都需選滿該彩種所規定的球數。)
- 提取自選號支援長按刪除自選號、單擊提取使用。
- 提取已儲存的自選號，直接帶入選號區供其他查詢操作。

#### 1.4 重複號碼分析
- 從歷史獎號中查出各彩種重複開出相同獎號資訊。
- **威力彩/大樂透**: 查詢重複六碼組合。
- **今彩539**: 查詢重複五碼組合。
- **三星彩**: 查詢重複三碼組合。
- **四星彩**: 查詢重複四碼組合。
- 顯示重複獎號及重複次數。
- 點選查出的某組號碼還可以查中獎期別、中獎日期等資訊。

#### 1.5 自選號中獎詳情查詢
- 根據自選號查詢所有中獎記錄(需選滿該彩種所要求的球數，威力彩第二區可選可不選)。
- 顯示中獎總筆數及按獎別統計中獎次數。
- 顯示中獎期別、開獎日期、獎號(自選號以不同顏色顯示)及獎別。
